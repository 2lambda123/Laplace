## Setup

```bash
pip install -r requirements.txt
# for development
pip install -e .
# for "production"
pip install .

# for the ASDL backend
git clone https://github.com/kazukiosawa/asdfghjkl.git
cd asdfghjkl
pip install -r requirements.txt
pip install .


# run tests
pip install pytest
pytest tests/
```
