## Setup

```bash
pip install -r requirements.txt
# for development
pip install -e .
# for "production"
pip install .

# for Kazuki's backend
cd dependencies
pip install asdfghjkl-0.0.1-py3-none-any.whl


# run tests
pip install pytest
pytest tests/
```
